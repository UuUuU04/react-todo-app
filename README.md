# Todoアプリ

Reactを用いて作成したシンプルなTodoアプリです。  
基本的なCRUD機能に加え、ユーザーの使いやすさを意識した機能を実装しました。

---

## 機能一覧
- タスクの追加・削除・編集（CRUD）
- 完了 / 未完了の切り替え（チェックボックス）
- フィルター機能（全て / 未完了 / 完了）
- ダークモード切り替え
- ドラッグ＆ドロップによる並び替え
- ローカルストレージによるデータ保存（リロード後も保持）

---

## 使用技術
- React（useState / useEffect）
- JavaScript（ES6）
- CSS

---

## 工夫した点

### ① 状態管理とコンポーネント設計
アプリ全体の状態（Todoリストや入力値など）はAppコンポーネントで一元管理し、  
入力欄・リスト・アイテムごとにコンポーネントを分割することで、  
コードがより読みやすく、修正しやすいようにしました。

---

### ② ユーザー体験（UX）の向上
- Enterキーでタスク追加できるようにし、操作の手間を削減
- ダークモードを実装し、利用環境に応じたUIを提供
- ドラッグ＆ドロップで直感的に並び替え可能に

---

### ③ データの永続化
useEffectを使用し、Todoリストの変更を検知してローカルストレージに保存することで、  
ページをリロードしてもデータが保持されるようにしました。

---

### ④ 実務を意識した実装
単なるTodoアプリに留まらず、  
フィルター機能や編集機能、並び替えなどを追加し、  
実際のアプリケーションに近い構成になるよう意識しました。
